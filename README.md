# 📊 Smart Analytics Dashboard

## 🚀 Overview

This project is a dynamic analytics dashboard built using Python, SQL, and Streamlit.

It supports:

* 📂 CSV Upload Analysis
* 🗄️ SQL Database Analysis
* 📊 Multiple Visualizations (Line, Bar, Pie, Histogram, Scatter)
* 🔍 Automatic Data Detection
* 🎯 Filtering & KPI Metrics

---

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit
* MySQL
* Matplotlib

---

## 📸 Features

* Upload any dataset and get instant insights
* Global chart selector
* Time series analysis
* Category-based analysis
* Clean UI dashboard

---

## ▶️ Run Locally

```bash
streamlit run app.py
```
